			Data base Initialization:
1. Inherit model the class BudgetInitializer from the class DropCreateDatabaseAlways
2. Run the application and log in as an admin (see the login and password into Seed method of the class BudgetInitializer)
3. Select the menu "Database initialization" and:
	a) init Account table
	b) init Ledger table

4.ATTENTION!!!!!!!!!!!!!
these actions removes data without ability to restore them!
To clear ledger  - select an action "Clear ledger"
The action "Clear account" removes all user's money accounts and connected ledger entries
